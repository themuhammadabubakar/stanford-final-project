# Within the lines - a coloring game

Within The Lines is an interactive digital coloring and painting application built with Python and Stanford’s graphics library. The app allows users to bring sketch illustrations to life using a realistic set of coloring tools designed to mimic the experience of coloring on paper. A vertical toolbar provides access to twelve colored pencils and an eraser, with the currently selected tool visually highlighted by “popping out” from the toolbar for easy identification. Users can freely paint within the coloring page using adjustable brush sizes controlled by intuitive plus and minus buttons, while a live circular cursor preview shows the exact size of the active brush. Painting is smooth and natural: users click to begin a stroke, move the cursor to draw, and click again to stop, with drawing automatically pausing when the cursor leaves the canvas and resuming when it returns. To preserve the appearance of traditional coloring books, the black sketch outlines are always rendered above the paint layer, ensuring crisp line art regardless of the colors applied underneath. The application also includes page management features that allow users to instantly switch to a randomly selected coloring page or clear everything and start fresh on a blank canvas. By combining creative freedom with an intuitive and visually engaging interface, Within The Lines provides a fun and accessible digital coloring experience for users of all ages.
